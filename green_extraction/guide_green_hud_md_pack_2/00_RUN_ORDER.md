# Codex 执行顺序说明（最终版，必须遵守）

## 项目目标
基于我提供的多张参考图，先提炼一套**原创可复用**的绿色 HUD / AR UI 风格规范，再实现一个**纯前端静态 HTML 页面**，用于预览与导出透明背景的 SVG / PNG 组件。

---

## 核心原则（必须）
1. **先风格学习，后编码实现**
2. **学习“风格语法”，不要复制具体图形/布局**
3. **导出 SVG / PNG 必须真实可用，优先级高于花哨效果**
4. **先完成最小可用版本（MVP），再扩展组件**
5. **所有输出文件按文件名分段给出，方便直接保存**

---

## 参考图说明（重要）
- 所有 `02251308_*.jpg` 仅用于风格提炼与结构参考
- 不允许复制参考图中的具体 icon path / 完整界面布局 / 原始文案数据
- 可以学习：
  - 线条粗细与圆角比例
  - HUD 边框语法
  - 信息层级与排版节奏
  - 绿色叠加透明感
  - 折线图/状态栏/提示条的视觉组织方式
