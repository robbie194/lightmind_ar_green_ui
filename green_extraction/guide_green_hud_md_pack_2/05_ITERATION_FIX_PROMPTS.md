## 模板 3：SVG/PNG 导出不稳定或背景不透明
请只修复导出功能相关问题，不要改 UI 视觉样式。

当前问题（按实际填写）：
- [ ] SVG 导出包含页面背景
- [ ] PNG 导出有黑底/白底
- [ ] PNG 导出尺寸不对
- [ ] glow 被裁切
- [ ] 导出按钮导出的不是目标组件本体
- [ ] 部分浏览器无法触发下载

修复要求：
1. 每个组件导出内容仅包含其内部 SVG
2. SVG 导出透明背景
3. PNG 导出透明背景，默认 2x
4. 正确处理 `viewBox` / `width` / `height`
5. 若使用 filter，确保 defs/filter 随组件导出且不被裁切
6. 若使用 Blob URL，导出后正确释放

请重点展示并注释：
- `exportSvg(...)`
- `exportPng(...)`
